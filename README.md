- v1.0 - 최초 배포 버전

## 버전 관리
sForm는 시맨틱 버전 지침을 따릅니다. 버전은 다음과 같은 형식을 따릅니다.

`<major>.<minor>.<patch>`

시멘틱버전에 관한 자세한 정보는 링크를 확인하시기 바랍니다. http://semver.org/.
